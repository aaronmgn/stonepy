    Clients¶

    Both clients are context managers and expose the same resource groups (see Resource groups). Use StoneXClient for synchronous code and AsyncStoneXClient for asyncio.

    stonepy.StoneXClient ¶

    StoneXClient(config: ClientConfig)

    Synchronous StoneX CIAPI v2 client.

    Use as a context manager or call close() when finished. Resource properties lazily construct typed API resource groups.
